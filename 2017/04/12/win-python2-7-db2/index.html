windows server 2008 r2 python2.7连接db2

发表于 2017-04-12 | 分类于 python | | 阅读次数

之前的老项目了，突然来个电话说要换服务器，要迁移程序，老项目使用的是db2 v7，版本，python2.7，用ibm_db模块连接db2,ibm_db依赖于db2的odbc的驱动，翻找了半条官方文档，发现驱动现在最低版本都到9.1了。。。
抱着试一试的态度在新服务器(windows server 2008 r2)上面折腾下

准备相关安装包和软件包

  • python2.7官方 python-2.7.13.amd64.msi
  • 微软提供的 VCForPython27.msi
  • python ibm_db模块 ibm_db-2.0.7.tar.gz
  • ibm官方odbc的驱动 nt32_odbc_cli.zip

安装步骤

  1. 直接运行 python-2.7.13.amd64.msi 安装即可
  2. 设置系统环境变量，让windows系统可以找的到python.exe, pip等命令
  3. 安装完成后，安装 VCForPython27.msi
  4. 解压 ibm_db-2.0.7.tar.gz 至 ibm_db-2.0.7 文件夹下
  5. 解压 nt32_odbc_cli.zip 至 ibm_db-2.0.7 下
  6. 进入 ibm_db-2.0.7 目录下运行python setup.py install

测试

经测试，可正常连接。万幸，万幸！
