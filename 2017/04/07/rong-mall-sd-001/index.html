荣远商城二开-部署文档

发表于 2017-04-07 | 分类于 荣远商城二开 | | 阅读次数

加密文档，仅供项目相关人员查看。

部署前准备

硬件

需要提供支持程序运行的相关硬件设施，可采用相关云服务商提供的服务器(阿里云，华为云)，配置要求如下：(带~的为以上)

类型 系统 CPU 内存 永久性存储 带宽
云服务器 (阿里云) centos7.2 2核~ 4G~ 硬盘:100G~ 2M~

域名

需提供一个经过备案后的域名，域名申请等可至 万网 等服务商注册备案

公众号

如需要接入公众号，需提供服务号，公众号申请入口

短信

需提供短信服务商的接口认证信息

第三方支付

需要准备银联，支付宝，微信，京东等支付接口相关的接口调用凭证

邮件

需提供邮件服务器，建议使用QQ企业邮箱

文字，图片素材准备

需要准备一下素材

  • 网站商城名称
  • 网站logo图片

安装系统

服务器安装运行环境

系统需要安装如下环境

  • 服务端语言: PHP5.4
  • web服务器: nginx1.10
  • 数据库: MySQL5

相关支持程序的安装，可以参考 ubuntu16.04 nginx php5.6-fpm环境搭建 进行部署安装

上传源码至服务器

用ftp上传工具或者其他方法将商城系统的源代码包拷贝至服务器

解压文件

将压缩包中的文件解压到至指定文件夹，如解压至:～/mall 文件夹下

修改nginx配置

修改～/mall/sconf路径下中的 nginx_mall.conf 文件中的root属性值，修改后的文件如下:

创建软链，使nginx生效

$ sudo ln -s ~/test/nginx_php_fpm.conf /etc/nginx/sites-enabled/
$ sudo nginx -s reload

没有任何问题，则访问 example.com 站点地址根据实际情况自行访问

访问后会出现安装引导页，根据安装引导页填写以下信息

  • 个人信息
  • 数据库信息

安装完成后，请将 ～/mall 路径下的 install 文件夹删除或者重命名

安装完成后，默认的后台地址为：example.com/admin

个性化配置

个性化配置请交与相关网站管理员进行设置

第三方服务配置

商城系统完成业务需要与第三方系统进行交互，准备好相关第三方的接口调用凭证后，交与网站管理员进行配置
