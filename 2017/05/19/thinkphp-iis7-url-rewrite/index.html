iis7安装配置url rewrite模块

发表于 2017-05-19 | 分类于 php | | 阅读次数

最近有个thinkphp的项目，因为通过URL重写隐藏应用的入口文件index.php，又把web server换成了iis7，所以，需要开启下iis7的url rewrite模块

准备

  • rewrite_amd64.msi 点击下载

安装

双击运行rewrite_amd64.msi，安装后打开iis7的管理界面，如果原来是打开的，重新打开一下，看到如下截图就代表安装成功

配置使用

  1. 找一个站点，双击网站下，站点的名字进入页面，点击URL Rewrite

  2. 右键点击Add Rule(s),随便先添加一个规则，为的是在原有的站点上生成web.config,如原有站点已有web.config文件，则可以直接编辑web.config文件
    )

  3. 编辑 web.config，如果你属性url rewrite 规则的话，可以自己编辑添加，不然，请查看相关框架的文档
    )
